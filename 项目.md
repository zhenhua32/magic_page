# 魔法页面修改器

做一个 chrome 插件, 主要功能是通过 ai 来修改页面上的内容, 让用户可以自定义页面的样式和内容.

1. 用于自定义接入openai 接口.
2. 通过对话的方式, 让用户输入想要修改的内容和样式.
3. 插件会根据用户的输入, 生成相应的 css 和 js 代码, 并注入到页面中.
4. 用户可以随时修改和删除注入的代码, 以达到想要的效果.
5. 插件还可以保存用户的设置, 以便下次使用时直接加载.
